# Tez Health App Icon Setup Instructions

## Step 1: Save Your Icon Image

**IMPORTANT:** Save the blue medical emergency icon image as:
```
E:\Flutter\Ramesh\tez_health\assets\images\tez_logo.png
```

### Requirements:
- **Format:** PNG (with transparent background preferred)
- **Size:** 1024x1024 pixels (minimum)
- **File name:** Exactly `tez_logo.png`

### How to save:
1. Right-click on the blue medical icon image
2. Select "Save Image As..."
3. Navigate to: `E:\Flutter\Ramesh\tez_health\assets\images\`
4. Save as: `tez_logo.png`

---

## Step 2: Generate App Icons

After saving the icon, open terminal/command prompt and run:

```bash
cd E:\Flutter\Ramesh\tez_health
flutter pub get
flutter pub run flutter_launcher_icons
```

This will automatically generate:
- ✅ All Android launcher icons
- ✅ Android adaptive icons
- ✅ All iOS app icons
- ✅ iOS dark mode variants

## Troubleshooting:

**If icons don't update:**
1. Uninstall the app completely from device
2. Run `flutter clean`
3. Run `flutter pub get`
4. Reinstall the app

**If you get "file not found" error:**
- Check that `tez_logo.png` exists in the exact path
- Verify the file name is correct (case-sensitive)
- Make sure it's PNG format
